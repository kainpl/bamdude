# Довідка API

BamDude надає REST API для інтеграції з зовнішніми інструментами та автоматизацією.

---

## :material-key: Автентифікація

### Автентифікація за API-ключем

Додайте API-ключ у заголовок `X-API-Key`:

```bash
curl -H "X-API-Key: your-api-key" \
  http://localhost:8000/api/v1/printers
```

### Отримання API-ключа

1. Перейдіть до **Налаштування** > **API-ключі**
2. Натисніть **Створити API-ключ**
3. Оберіть дозволи
4. Скопіюйте ключ (показується лише один раз)

---

## :material-web: Інтерактивний браузер API

BamDude включає вбудований браузер API у **Налаштування** > **API-ключі** для дослідження та тестування ендпоінтів.

- Групування за категоріями (принтери, архіви, черга тощо)
- Автоматично заповнені приклади тіла запиту
- Виконання в реальному часі з форматованими відповідями

---

## :material-api: Основні ендпоінти

### Принтери

| Метод | Ендпоінт | Опис |
|-------|----------|------|
| GET | `/api/v1/printers` | Список усіх принтерів |
| POST | `/api/v1/printers` | Додати принтер |
| GET | `/api/v1/printers/{id}` | Отримати деталі принтера |
| GET | `/api/v1/printers/{id}/status` | Отримати статус принтера |

### Архіви

| Метод | Ендпоінт | Опис |
|-------|----------|------|
| GET | `/api/v1/archives` | Список архівів |
| GET | `/api/v1/archives/{id}` | Отримати деталі архіву |
| POST | `/api/v1/archives/{id}/reprint` | Передрукувати архів |

### Черга

| Метод | Ендпоінт | Опис |
|-------|----------|------|
| GET | `/api/v1/queue` | Отримати статус черги |
| POST | `/api/v1/queue` | Додати до черги |
| DELETE | `/api/v1/queue/{id}` | Видалити з черги |

### Бібліотека

| Метод | Ендпоінт | Опис |
|-------|----------|------|
| GET | `/api/v1/library/files` | Список файлів |
| POST | `/api/v1/library/files/upload` | Завантажити файл |

### Метрики

| Метод | Ендпоінт | Опис |
|-------|----------|------|
| GET | `/api/v1/metrics` | Метрики Prometheus |

## :material-code-json: Схема OpenAPI

Повна схема OpenAPI доступна за адресою:

```
GET /api/v1/openapi.json
```

Використовуйте її зі Swagger UI, Postman або будь-яким OpenAPI-сумісним інструментом.

> Початково базується на документації [Bambuddy](https://github.com/maziggy/bambuddy).
