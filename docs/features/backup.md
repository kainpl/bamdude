# Backup & Restore

Protect your print history with database backups and restore when needed.

---

## :material-backup-restore: Overview

BamDude's backup system:

- **Full database backup** -- All data in one file
- **User settings included** -- Preferences preserved
- **Archives included** -- Print history saved
- **ZIP format** -- Includes 3MF files and thumbnails when selected

---

## :material-download: Creating a Backup

1. Go to **Settings** > **Backup & Restore**
2. Click **Create Backup**
3. Choose what to include (database, archives, 3MF files)
4. Download the backup file

---

## :material-upload: Restoring a Backup

1. Go to **Settings** > **Backup & Restore**
2. Click **Restore**
3. Upload your backup file
4. Confirm the restore

!!! danger "Data Loss"
    Restoring a backup replaces all current data. Create a backup of your current state first.

---

## :material-lightbulb: Tips

!!! tip "Regular Backups"
    Schedule regular backups, especially before upgrading BamDude.

!!! tip "Off-Site Storage"
    Store backups on a different machine or cloud storage for disaster recovery.
