# Режим заміни пластин

Режим заміни пластин забезпечує підтримку механізмів заміни пластин A1 Mini -- автоматизованих систем, що змінюють робочі пластини між друками для безперервного виробництва без ручного втручання.

---

## :material-swap-horizontal: Що таке режим заміни пластин?

Механізм заміни пластин -- це механічний додаток для A1 Mini, який автоматично виштовхує готову робочу пластину та встановлює нову. Коли режим заміни увімкнено в BamDude, планувальник черги координується з механізмом для:

1. Запуску **swap-файлу** (G-code) після завершення кожного друку
2. Виклику **макросів** у потрібні моменти
3. Автоматичного запуску наступного друку з черги

---

## :material-cog: Налаштування

### Увімкнення режиму заміни

1. Перейдіть до **Settings > Queue**
2. Увімкніть **Swap Mode** для принтера A1 Mini
3. Налаштуйте шлях до swap-файлу (файл G-code на SD-картці принтера)

### Swap-файл

Swap-файл -- це файл G-code, який інструктує принтер виштовхнути поточну пластину та підготуватися до наступного друку. Типові операції:

- Переміщення стола вперед у позицію виштовхування
- Очікування спрацювання механізму заміни
- Повернення осей у початкове положення
- Підготовка до наступного друку

```gcode
; Example swap file
G28 X Y         ; Home X and Y
G1 Y 180 F3000  ; Move bed forward for plate swap
M400            ; Wait for moves to complete
G4 S5           ; Pause 5 seconds for swap
G28             ; Home all axes
```

!!! warning "Потрібне спеціальне обладнання"
    Режим заміни пластин потребує фізичного механізму заміни, встановленого на A1 Mini. Swap-файл має бути адаптований до вашого конкретного обладнання.

---

## :material-code-braces: Інтеграція з макросами

Режим заміни працює разом із [системою макросів](macros.md) для вставки G-code у ключових точках:

| Подія | Коли спрацьовує | Типове використання |
|-------|----------------|---------------------|
| **Print End** | Після завершення друку | Виштовхування пластини |
| **Print Start** | Перед початком наступного друку | Повернення осей, перевірка пластини |

Налаштуйте макроси у **Settings > Macros** з подіями, специфічними для режиму заміни.

---

## :material-playlist-play: Поведінка черги в режимі заміни

Коли режим заміни активний:

1. Друк завершується на A1 Mini
2. **Swap-файл** виконується автоматично
3. **Підтвердження очищення пластини пропускається** -- механізм заміни бере це на себе
4. Наступний друк з черги запускається автоматично
5. Цикл повторюється, поки черга не спорожніє

Це дозволяє **автономне пакетне виробництво** на принтерах A1 Mini.

## :material-lightbulb: Поради

!!! tip "Протестуйте swap-файл"
    Запустіть swap-файл вручну (з файлового браузера принтера) перед увімкненням режиму заміни, щоб переконатися, що він коректно працює з вашим обладнанням.

!!! tip "Поєднайте з пакетною кількістю"
    Використовуйте функцію пакетної кількості для додавання кількох копій у чергу, а потім дозвольте режиму заміни виконати їх одну за одною автоматично.

!!! tip "Віддалений моніторинг"
    Використовуйте потокове відео з камери та Telegram-бот для віддаленого моніторингу операцій заміни та отримання сповіщень про завершення черги.
