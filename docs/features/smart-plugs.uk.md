# Розумні розетки

Керуйте принтерами за допомогою розумних розеток Tasmota, Home Assistant, REST/Webhook або MQTT для моніторингу живлення, автоматизації та відстеження енергоспоживання.

---

## :material-power-plug: Огляд

Інтеграція з розумними розетками забезпечує:

- **Управління живленням** -- дистанційне увімкнення/вимкнення принтерів
- **Моніторинг енергії** -- відстеження споживання електроенергії
- **Автоувімкнення** -- запуск принтера перед запланованим друком
- **Автовимкнення** -- вимкнення після охолодження

---

## :material-cog: Підтримувані типи

| Тип | Управління | Енергія | Опис |
|-----|:----------:|:-------:|------|
| **Tasmota** | :material-check: | :material-check: | Пряме управління розетками з прошивкою Tasmota |
| **Home Assistant** | :material-check: | :material-check: | Будь-яка сутність switch/light через HA |
| **REST / Webhook** | :material-check: | :material-check: | Довільні HTTP API ендпоінти |
| **MQTT** | :material-close: | :material-check: | Тільки моніторинг енергоспоживання |

---

## :material-robot: Автоматизація

### Автоувімкнення

Коли друк з черги готовий до запуску, BamDude вмикає розетку, чекає на завантаження принтера, а потім починає друк.

### Автовимкнення

Після завершення друку BamDude чекає охолодження стола, перевіряє наявність інших друків у черзі, а потім вимикає живлення.

Налаштування знаходяться у **Settings** > **Smart Plugs** з параметрами температури та часу охолодження.

---

## :material-lightbulb: Поради

!!! tip "Почніть просто"
    Спочатку використовуйте ручне управління живленням, перш ніж вмикати автоматизацію.

!!! tip "Перевірте охолодження"
    Відстежте кілька друків, щоб знайти оптимальну температуру охолодження для вашого принтера.

> Базується на документації [Bambuddy](https://github.com/maziggy/bambuddy).
